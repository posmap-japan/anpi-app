Anpi（あんぴ）

利用規約

最終更新日: 2026年2月19日

この利用規約（以下「本規約」）は、Anpi（以下「本アプリ」）の利用に関する条件を定めるものです。本アプリをご利用いただく前に、本規約をよくお読みください。本アプリを利用することにより、本規約に同意したものとみなします。

第1条（サービスの概要）

本アプリは、大切な人の安否を確認するための見守りサービスです。主な機能として以下を提供します。

  • 見守り対象者の安否状態の確認
  • 安否確認メッセージの送受信
  • 見守り時間の設定と通知
  • プッシュ通知による安否アラート

第2条（アカウント）

  1. 本アプリの利用には、Apple ID または Google アカウントによるサインインが必要です。
  2. ユーザーは、自身のアカウント情報を適切に管理する責任を負います。
  3. アカウントの不正利用が判明した場合、当方はアカウントを停止する権利を有します。

第3条（料金プラン）

  1. 本アプリは、無料プランと有料サブスクリプションプランを提供します。
  2. 無料プランでは1人までの見守りが永続的に利用可能です。
  3. 有料プランの料金は以下の通りです。
    • ライトプラン: 最大2人まで（月額¥390 / 年額¥3,900）
    • ファミリープラン: 最大10人まで（月額¥980 / 年額¥9,800）
    • ビジネスプラン: 最大50人まで（月額¥4,000 / 年額¥40,000）
  4. サブスクリプションは自動更新されます。更新日の24時間前までに解約しない限り、自動的に更新されます。
  5. サブスクリプションの管理・解約は、Apple IDの設定から行えます。
  6. 購入後の返金は、Appleの返金ポリシーに従います。

第4条（禁止事項）

ユーザーは、以下の行為を行ってはなりません。

  • 法令または公序良俗に反する行為
  • 他のユーザーの同意なく見守り機能を悪用する行為
  • 本アプリの運営を妨害する行為
  • 不正アクセスまたはそれを試みる行為
  • 他のユーザーへの嫌がらせやストーカー行為
  • 本アプリを逆アセンブル、逆コンパイル、その他の方法で解析する行為
  • その他、当方が不適切と判断する行為

第5条（免責事項）

  1. 本アプリは「現状のまま」提供されます。安否確認の正確性や即時性を完全に保証するものではありません。
  2. 本アプリは、緊急通報サービス（110番、119番等）の代替ではありません。緊急時は必ず適切な緊急サービスにご連絡ください。
  3. ネットワーク障害、サーバー障害、デバイスの問題等により、通知が遅延または届かない場合があります。
  4. 本アプリの利用により生じた損害について、当方は法令で認められる範囲で責任を負いません。

第6条（サービスの変更・終了）

  1. 当方は、事前の通知なく本アプリの内容を変更、追加、または削除することがあります。
  2. 当方は、やむを得ない事由がある場合、本アプリの提供を一時的または永続的に停止することがあります。

第7条（知的財産権）

本アプリに関するすべての知的財産権は、当方に帰属します。ユーザーは、当方の事前の書面による承諾なく、本アプリの内容を複製、転載、改変することはできません。

当方は、必要に応じて本規約を変更できるものとします。変更後の利用規約は、本ページに掲載した時点から効力を持つものとします。

第9条（準拠法・管轄裁判所）
